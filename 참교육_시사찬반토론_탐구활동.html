프로그램 개요

한국경제 생글생글의 시사 찬반토론 주제를 활용한 고등학교 탐구활동 도구입니다. AI 없이 학생 스스로 단계별 활동지를 완성하도록 설계되었습니다.

수업 활용 방법

  1. 개인 탐구 — 학생이 관심 주제를 선택하여 혼자 활동지 완성
  2. 모둠 토론 — 같은 주제로 찬성/반대 팀을 나눠 토론 후 정리
  3. 발표 수업 — 발표 슬라이드 기능으로 바로 발표로 연결
  4. 포트폴리오 — HTML 저장 기능으로 탐구 결과물 보관

5단계 탐구 활동 안내

  1. 1단계 주제 파악 — 배경지식, 핵심 키워드, 탐구 전 질문 확인
  2. 2단계 찬반 분석 — 찬성·반대 논거 분석 및 추가 논거 직접 정리
  3. 3단계 탐구 설계 — 탐구 주제 선정, 방법 계획 수립
  4. 4단계 내 의견 — 입장 표명, 근거 서술, 반론 대응
  5. 5단계 마무리 — 체크리스트 점검, 소감 작성

저장 및 출력 안내

  • 💾 저장 — 작성 내용을 HTML 파일로 다운로드 (학생 이름 파일 관리 권장)
  • 📂 불러오기 — 저장된 HTML 파일 업로드하여 이어서 작성
  • 🖨 PDF — 브라우저 인쇄 기능으로 PDF 저장 (Ctrl+P 또는 Cmd+P)
  • 🎬 발표 — 작성 내용 기반 발표 슬라이드 자동 생성, 키보드 방향키로 전환
💡 팁: 수업 전 관련 생글생글 기사를 미리 읽어오면 더 깊이 있는 탐구가 가능합니다. 각 주제의 🔗 링크로 원문 기사를 확인하세요.

📖 학생 설명서

탐구활동이란?

시사 이슈에 대해 찬성과 반대 의견을 살펴보고, 나만의 주장을 논리적으로 정리하는 활동입니다. 한국경제 생글생글에 실린 최신 주제들로 진행해요!

이렇게 하세요!

  1. 왼쪽에서 관심 있는 주제를 클릭하세요
  2. 주제 카드의 🔗 아이콘으로 원문 기사를 읽어보세요
  3. 1단계부터 5단계까지 순서대로 활동지를 채우세요
  4. 다 쓰면 💾 저장으로 내 파일을 보관하세요
  5. 🎬 발표 버튼으로 발표 슬라이드를 바로 만들 수 있어요

원문 기사 안내

각 주제의 🔗 아이콘을 클릭하면 한국경제 생글생글 홈페이지의 관련 기사로 이동합니다. 기사를 읽으면 더 풍부한 탐구를 할 수 있어요!

이어서 쓰고 싶다면?

💾 저장 → HTML 파일 다운로드 → 나중에 📂 불러오기로 열면 이전에 쓴 내용이 그대로 복원돼요.

💡 팁: 먼저 내 생각을 써보고 원문 기사를 읽으면, 내 생각이 어떻게 달라지는지 비교해볼 수 있어요!
